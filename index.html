À propos de ManLore

ManLore est votre compagnon ultime pour gérer et organiser votre collection de bandes dessinées numériques. Que vous soyez fan de mangas, manhwas, manhuas, webtoons, light novels, webnovels ou webcomics, ManLore vous offre une plateforme complète pour cataloguer, suivre et apprécier vos lectures préférées.

Conçue avec une interface moderne et intuitive, l'application reprend toutes les fonctionnalités du site web ManLore dans une expérience mobile optimisée. Gérez votre bibliothèque personnelle en quelques clics, suivez vos progrès de lecture, notez vos titres favoris et ne perdez plus jamais le fil de vos histoires en cours.

Avec la synchronisation cloud automatique, toutes vos données sont sauvegardées en temps réel. Que vous utilisiez l'application sur votre téléphone, votre tablette ou le site web sur votre ordinateur, vos collections, notes et statistiques restent parfaitement synchronisées. Plus de risque de perdre vos précieuses informations !

Installation

Important : ManLore n'est pas disponible sur le Google Play Store. Vous devez autoriser l'installation d'applications provenant de sources inconnues.
  1. Téléchargez le fichier manlore.apk en cliquant sur le bouton ci-dessus
  2. Ouvrez les paramètres de votre appareil Android
  3. Allez dans Sécurité ou Applications
  4. Activez l'option "Sources inconnues" ou "Installer des applications inconnues" pour votre navigateur
  5. Localisez le fichier manlore.apk téléchargé dans vos téléchargements
  6. Appuyez sur le fichier pour lancer l'installation
  7. Suivez les instructions à l'écran pour finaliser l'installation
  8. Lancez ManLore et connectez-vous avec votre compte existant ou créez-en un nouveau
Astuce : Une fois l'installation terminée, vous pouvez désactiver l'option "Sources inconnues" pour plus de sécurité.
